# 🍽️ Nutrition Tracker (Full Stack)

A full-stack nutrition tracking web application to monitor daily calorie intake, macros, and weight progress.

---

## 🚀 Features

- Track **Calories, Protein, Carbs, Fat**
- 📊 Progress chart (Chart.js)
- 🔥 Daily streak tracking
- 🧠 Insights (best day, average intake, status)
- ⚖️ Weight tracking with history
- 🗑️ Add / Delete entries
- 🌙 Dark mode support

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Chart.js
- CSS (Custom styling)

### Backend
- Node.js
- Express.js

---

## 📂 Project Structure

nutrition-tracker/
│
├── calorie-tracker-app/ # Frontend (React)
├── nutrition-backend/ # Backend (Node + Express)


---

## ⚙️ How to Run Locally

### 1. Clone repo

git clone https://github.com/Vetrivel2505/nutrition-tracker.git

cd nutrition-tracker


### 2. Run backend

cd nutrition-backend
npm install
node server.js


### 3. Run frontend

cd ../calorie-tracker-app
npm install
npm run dev
